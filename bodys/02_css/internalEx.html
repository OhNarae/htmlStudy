2. ** CSS Internal **

내부 스타일 시트(Internal Style Sheet)
내부 스타일 시트는 하나의 문서에서만 사용하는 스타일을 정의할 때 사용합니다.
내부 스타일은 head 태그 안에 style 태그를 이용하여 정의합니다.

브라우저는 알려지지 않는 태그를 무시합니다.
스타일을 지원하지 않는 오래된 버전의 브라우저일 경우 style 태그를 무시하고
style 태그 안의 내용을 웹 페이지에 출력하게 됩니다.
