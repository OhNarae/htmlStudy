** CSS에서 많이 사용되는 속성들 **

color : 텍스트 색상
font-weight : 볼드체 설정
font-size : 폰트 크기
font-style : 이탤릭 설정
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
padding : 요소의 가장자리와 내용사이의 간격
margin : 요소의 가장자리와 그 밖에있는 다른 요소와의 간격
border : 요소 감싸고 있는 외부의 경계선
background-image : 배경 이미지

background-color : 배경색
text-align : 텍스트 정렬
